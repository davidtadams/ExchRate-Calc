Currency Converter

Pick your currencies to convert and see the results.

The site will also show you a currency exchange rate table for the currency you choose.

All of the exchange rate data has been pulled from the API available from fixer.io The data is updated on a daily basis, so it is not real time. Data only available for 32 listed currencies in the dropdown menu. Created as a mini-project on 11/9/15 while in gSchool! View it on GitHub! with a little bit of design critique from alexG53090
